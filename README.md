# SNAKE

A simple snake game.

## Demo

You can play the game [here](https://snake.albenesi.us).

## Usage

1. Install webpack by running `npm install` or `yarn install`.
2. Build by running `npm run build` or `yarn run build`.
3. Open `dist/index.html` in a browser.
